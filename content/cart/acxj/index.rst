Pokémon Card GB
===============

:lang: en
:slug: cart/acxj
:tags: cart

* Release Date: 1998-12-18
* Product Code: DMG-ACXJ-JPN
* Mapper_: HuC-1
* ROM Size: 8Mbit (1MiB)
* RAM Size: 256Kbit (32KiB)

The Japanese version of the Pokémon Trading Card Game cartridge is a HuC-1 game, supporting infrared connectivity but not GBKiss. The international version instead use standard MBC-5 cartridges and support infrared connectivity only on the Game Boy Color.
