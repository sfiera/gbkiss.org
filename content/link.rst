GBKiss Link
===========

:lang: en
:slug: link

* Original Title: ジービーキスリンク
* Release Date: 1998-03-06
* Release Price: ¥4,980
* Platform: Windows 95

Sold as a package containing the modem and two floppies: one with the modem driver and one with 29 software files for transfer.
